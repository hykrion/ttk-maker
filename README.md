# ttk-maker / ttk-cloner

With this program you can clone the fantastic themes [Azure](https://github.com/rdbende/Azure-ttk-theme) and [Forest](https://github.com/rdbende/Forest-ttk-theme) from rdbende. To do this you only have to select the name of the new theme and its main color. Then copy the 'light' and 'dark' directories that will have been created in the directory of your new theme and you can now run 'example.tcl azure/forest light/dark' to see the results.
